# Prog-part-1

Register and Login System
This project is a Java console application that allows a user to register and log in.

Features

Checks if the username has an underscore and is no more than 5 characters.
Checks if the password has:
At least 8 characters
A capital letter
A number
A special character

Checks if the cell phone number is in South African format (+27 followed by 9 digits).
Allows the user to register and then log in.

Key Features
Real time messaging
Multi-Client support
Uer Friendly 

Requirements
Java Developent Kit
Any Java IDE 

How to Run

Open the project in NetBeans (or any Java IDE).
Run the file RegisterandLogin.java.
Follow the menu:
Press 1 to Register
Press 2 to Login
Press 3 to Exit
