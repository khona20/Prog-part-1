Register and Login System

Features

Checks if the password has:
At least 8 characters

Allows the user to register and then log in.

Register and Login App with Message Management

A Java application that allows users to register, login, and manage messages. The app stores sent, disregarded, and stored messages, along with their associated metadata such as message ID, message hash, sender, recipient, and message content.

Features:

- User registration and login functionality
- Message management (add, display, search, delete)
- Message categorization (sent, disregarded, stored)
- Message metadata storage (message ID, message hash, sender, recipient, message content)

Technologies Used:

- Java
- ArrayList for data storage

API Documentation

The Chat App provides the following API methods:

- registerUser(String name, String email, String password): Registers a new user with the given name, email, and password.
- loginUser(String email, String password): Logs in a user with the given email and password.
- addMessage(String message, String status): Adds a new message with the given text and status (sent, disregarded, or stored).
- displaySentMessages(): Displays all sent messages.

These methods allow you to interact with the Chat App's features, such as user registration, login, and message management.

Features

The Chat App includes the following features:

- User Registration and Login: Users can register and log in to the app.
- Message Management: Users can add, display, search, and delete messages.
- Message Categorization: Messages are categorized as sent, disregarded, or stored.
- Message Metadata Storage: The app stores message metadata, including message ID, message hash, sender, recipient, and message content.

Example Use Cases:

- Register a new user
- Login to the app
- Add a new message
- Display sent messages
- Search for a message by ID or recipient
- Delete a message by hash
